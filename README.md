# Tic-Tac-Toe: Human vs AI

An unbeatable Tic-Tac-Toe web application featuring an AI opponent powered by the Minimax algorithm with Alpha-Beta pruning.

### Installation and Setup

To run this project, you must have Python installed. Use the following bash commands to install dependencies and start the game:

```bash
# Install required libraries
pip install streamlit numpy

# Run the application
streamlit run tic-tac-toe.py
